[size=4][color=#bf307a]Introduction[/color][/size]
[size=1em]Team Page lets you add multiple custom pages to a team/staff page with groups, moderators or text.[/size]

[size=3][color=#1f4882]Features[/color][/size]
[list]
[li][size=0.9em]Enable Team Page[/size][/li]
[li][size=0.9em]Create different pages to show in the Team Page action[/size]
[list]
[li][size=0.8em]Groups page[/size]
[list]
[li][size=0.8em]Sortable: Drag and drop management[/size][/li]
[/list]
[/li]
[li][size=0.8em]Moderators page[/size]
[list]
[li][size=0.8em]Select the style of the blocks (boards or users based)[/size][/li]
[li][size=0.8em]Select the boards included for the page[/size][/li]
[/list]
[/li]
[li][size=0.8em]BBC Text Page[/size][/li]
[li][size=0.8em]HTML Text Page[/size][/li]
[li][size=0.8em]Subaction name for the custom page[/size][/li]
[/list]
[/li]
[li][size=0.9em]Permissions to view the Team Page[/size][/li]
[li][size=0.9em]Layout/view options[/size]
[list]
